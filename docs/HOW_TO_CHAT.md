# How to Use Chat

This explains how the chat feature works. It's pretty straightforward once you get the hang of it.

## Quick Start

1. Click "Chat" in the navbar
2. Click the search icon (ğŸ”)
3. Search for a user
4. Click on them
5. Type a message and press Enter

That's it!

## Step by Step

### Starting a Chat

**Method 1: From Chat Page**

1. Go to "Chat" page
2. Click the magnifying glass icon (ğŸ”) at the top
3. Type username or email
4. Click "Search"
5. Click on the user you want
6. Chat window opens on the right

**Method 2: From Friends Page**

1. Go to "Friends"
2. Find the person
3. Click "Chat" button
4. Takes you to Chat page with that conversation

### Sending Messages

1. Select a chat (or start a new one)
2. Type in the message box at the bottom
3. Press Enter or click Send
4. Your message appears on the right (blue)
5. Their reply appears on the left (gray)

## Understanding the Interface

### Chat Sidebar (Left)

Shows all your conversations:
- **Header**: "Chats" title and search icon
- **Filters**: All, Unread, Prefered, Groups
- **Chat List**: All your conversations
  - Shows avatar, name, last message preview
  - Green dot = online
  - Star icon = mark as preferred

### Chat Window (Right)

Where you see messages:
- **Header**: Other person's name, online status
- **Messages**: Your messages (right, blue), theirs (left, gray)
- **Input**: Type here, emoji button, attach button, send button

### Filters

At the top of sidebar:
- **All**: All chats
- **Unread**: Only unread messages
- **Prefered**: Only starred chats
- **Groups**: Only group chats

## How It Works

- Chat rooms are created automatically when you send the first message
- Messages are real-time (instant)
- Typing indicators show when someone is typing
- Messages are saved so you can see history later

## Troubleshooting

**"No chats yet"**
- You haven't started any conversations
- Click search icon and find someone to chat with

**Can't find user**
- Check spelling
- Make sure user exists
- Try partial username search

**Messages not sending**
- Check internet connection
- Make sure you're logged in
- Refresh the page
- Check if backend is running

**Chat not appearing**
- Make sure you actually sent a message (not just opened chat)
- Refresh the page
- Check if message appears in chat window

**Messages not real-time**
- Refresh page
- Check browser console (F12) for errors
- Make sure backend is running

## Tips

- Search works with partial usernames
- Mark important chats as preferred
- Use filters to find unread messages
- Works on mobile too

**Q: How many people in a group?**
